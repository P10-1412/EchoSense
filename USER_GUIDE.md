# EchoSense 新功能使用指南

## 1. 目标群体画像设置

### 如何使用
1. 点击页面顶部的"设置"按钮
2. 滚动到"目标群体画像"部分
3. 填写以下信息：
   - **人口统计特征**：描述您的目标受众的年龄、地域、学历等
   - **兴趣爱好**：列出受众感兴趣的主题和领域
   - **痛点需求**：描述受众面临的问题和需求
   - **消费习惯**：说明受众的消费能力和偏好
   - **媒体偏好**：列出受众常用的媒体平台
4. 点击"保存设置"按钮

### 效果
- AI分析时会根据您填写的目标群体画像，提供更精准的商业化建议、传播策略和风险评估
- 建议内容会更贴合您的实际受众特征

### 示例
```
人口统计特征：25-35岁职场人士，一二线城市，本科及以上学历
兴趣爱好：职场发展、个人成长、知识学习
痛点需求：职业瓶颈、时间管理、技能提升
消费习惯：愿意为优质内容付费，注重性价比
媒体偏好：播客、视频号、公众号、小红书
```

## 2. 一键重置功能

### 如何使用
1. 点击页面顶部的"一键重置"按钮（红色文字）
2. 在弹出的确认对话框中点击"确定"

### 注意事项
⚠️ **此操作不可恢复！** 将清除以下所有数据：
- 用户画像数据
- 系统设置（包括目标群体画像）
- 历史分析记录
- 案例数据库
- 当前分析结果
- 输入内容

### 适用场景
- 想要重新开始，清除所有历史数据
- 切换到新的账号或项目
- 测试系统默认状态

## 3. 品牌更新说明

### choSense → EchoSense
- 项目已更名为 **EchoSense**
- 所有界面文字已更新
- 您的历史数据会自动迁移，无需手动操作

### 数据迁移
系统会自动将旧版本（choSense）的数据迁移到新版本（EchoSense），包括：
- 用户画像
- 系统设置
- 历史记录
- 案例数据库

迁移过程完全自动，您无需进行任何操作。

## 常见问题

### Q: 目标群体画像必须填写吗？
A: 不是必须的，但强烈建议填写。填写后AI分析会更精准，建议会更贴合您的实际情况。

### Q: 可以随时修改目标群体画像吗？
A: 可以。随时进入设置修改，修改后的画像会在下次分析时生效。

### Q: 一键重置后能恢复数据吗？
A: 不能。重置操作会永久删除所有数据，请谨慎使用。

### Q: 旧版本的数据会丢失吗？
A: 不会。系统会自动将choSense的数据迁移到EchoSense，您的所有历史数据都会保留。
