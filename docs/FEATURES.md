# choSense 功能详解

## 一、量化指标系统

### 1. 四大核心指标

#### 💰 金钱价值（Money Value）
- **用途**：量化商业收益和潜在损失
- **应用场景**：
  - 广告位预估收益：1000-50000元
  - 品牌合作收入预估
  - 法律风险罚款预估
  - 内容侵权损失预估
- **示例**：
  - "第15分钟情绪高峰适合植入广告，预估收益 ¥8,000"
  - "第22分钟敏感言论存在法律风险，潜在罚款 ¥20,000"

#### 👥 粉丝影响（Fans Impact）
- **用途**：量化传播影响力和掉粉风险
- **应用场景**：
  - 传播爆点影响人数：500-100000人
  - 观点共鸣预估触达
  - 舆论风险掉粉预估
  - 争议话题影响范围
- **示例**：
  - "第8分钟观点具有强传播性，预估影响 15,000 粉丝"
  - "第18分钟言论可能引发争议，预估掉粉 3,000 人"

#### 📊 互动率（Engagement Rate）
- **用途**：量化内容互动潜力
- **应用场景**：
  - 点赞率提升预估：1-20%
  - 评论互动增长预估
  - 转发传播率预估
  - 社交媒体热度预估
- **示例**：
  - "第10分钟话题具有强互动性，预估互动率提升 +8%"
  - "优化开场白后，预估整体互动率提升 +5%"

#### ⭐ 品牌价值（Brand Value）
- **用途**：量化品牌形象影响
- **评分范围**：0-100分
- **应用场景**：
  - 正面形象塑造：70-95分
  - 专业度提升评估
  - 品牌调性匹配度
  - 负面影响预警：50-69分
- **示例**：
  - "专业观点分享提升品牌专业形象，品牌价值 85/100"
  - "情绪化表达可能损害品牌形象，品牌价值 62/100"

## 二、四类智能建议

### 1. 💰 广告位机会（Ad Placement）

**识别标准**：
- 情绪高峰点（兴奋、共鸣、认同）
- 话题转折点
- 观众注意力集中时刻
- 内容自然过渡点

**输出内容**：
```
位置：第15分钟30秒 - 第16分钟15秒
标题：情绪高峰广告植入机会
描述：讨论创业成功案例时观众情绪达到高峰，适合植入相关产品
预估价值：
  - 金钱：¥8,000
  - 品牌价值：82/100
详细建议：
  在讨论创业成功的关键因素后，自然过渡到产品介绍，
  强调产品如何帮助创业者提升效率。
口播文案示例：
  "说到提升效率，我最近一直在用XX工具，真的帮我节省了
  很多时间。如果你也在创业路上，可以试试看..."
```

### 2. 🚀 传播爆点（Viral Point）

**识别标准**：
- 独特观点或反常识内容
- 情绪共鸣强烈的话题
- 社会热点相关讨论
- 具有争议性但安全的话题

**输出内容**：
```
位置：第8分钟 - 第9分钟30秒
标题：反常识观点传播爆点
描述：提出"失败比成功更重要"的反常识观点，具有强传播性
预估价值：
  - 粉丝影响：15,000人
  - 互动率：+8%
  - 品牌价值：88/100
详细建议：
  这个观点具有强烈的话题性，建议制作短视频切片，
  配合文案"你以为的成功，可能是最大的失败"进行传播。
推广策略：
  1. 制作15秒短视频，突出核心观点
  2. 在微博、抖音、小红书同步发布
  3. 使用话题标签 #创业真相 #反常识思维
  4. 引导评论区讨论，增加互动
  5. 预计3天内达到10万播放量
```

### 3. ⚠️ 风险预警（Risk Warning）

**识别标准**：
- 敏感话题或言论
- 可能引发争议的观点
- 法律法规边界内容
- 可能导致掉粉的表达

**输出内容**：
```
位置：第22分钟15秒 - 第23分钟
标题：敏感言论舆论风险
描述：对某行业的负面评价可能引发该行业从业者反感
预估风险：
  - 潜在掉粉：3,000人
  - 品牌价值：-15分
  - 潜在损失：¥5,000（公关处理成本）
风险原因：
  1. 表达过于绝对，缺乏客观性
  2. 未考虑该行业从业者的感受
  3. 可能被断章取义传播
  4. 与品牌定位的包容性相悖
修改建议：
  原话："这个行业就是在割韭菜"
  修改为："这个行业确实存在一些不规范的现象，
  但也有很多从业者在努力改变，我们应该理性看待"
  
  修改要点：
  - 避免绝对化表达
  - 增加客观性和平衡性
  - 承认行业多样性
  - 保持理性和包容的态度
```

### 4. 💡 内容优化（Content Optimization）

**识别标准**：
- 可以改进的表达方式
- 结构优化机会
- 信息密度提升空间
- 观众体验改善点

**输出内容**：
```
位置：开场白（0-2分钟）
标题：开场白优化建议
描述：当前开场白过于冗长，建议精简并增加悬念
预估提升：
  - 互动率：+5%
  - 品牌价值：+8分
详细建议：
  当前开场白用时2分钟，包含过多寒暄和铺垫。
  建议：
  1. 前30秒直接抛出核心话题
  2. 用一个引人入胜的故事或数据开场
  3. 在1分钟内进入主题
  4. 将个人介绍移至中段自然穿插
  
  优化后预期效果：
  - 降低前3分钟流失率20%
  - 提升整体完播率15%
  - 增强专业形象感知
```

## 三、智能阈值系统

### 阈值设置界面

用户可以在"阈值设置"对话框中自定义四个指标的触发值：

```
💰 金钱价值阈值：[输入框] 元
   说明：广告位、商业合作等预估收益超过此金额时弹窗提示
   默认值：5000元
   建议范围：1000-50000元

👥 粉丝影响阈值：[输入框] 人
   说明：预估影响粉丝数或潜在掉粉风险超过此数量时弹窗提示
   默认值：1000人
   建议范围：500-100000人

📊 互动率阈值：[输入框] %
   说明：预估互动率提升超过此百分比时弹窗提示
   默认值：5%
   建议范围：1-20%

⭐ 品牌价值阈值：[输入框] 分
   说明：品牌形象提升指数超过此分数时弹窗提示
   默认值：70分
   建议范围：50-95分
```

### 触发逻辑

当任一建议的任一指标超过对应阈值时，该建议会被标记为"高价值"：

```typescript
// 示例：某条广告位建议
{
  value: {
    money: 8000,      // 超过阈值5000 ✓
    brandValue: 82    // 超过阈值70 ✓
  }
}
// 结果：触发高价值弹窗
```

### 弹窗展示

高价值弹窗会：
1. 自动弹出，无需用户操作
2. 显示所有超过阈值的建议
3. 按优先级排序
4. 突出显示超过阈值的具体指标
5. 提供"查看全部建议"按钮

## 四、使用场景示例

### 场景1：商业化变现
**用户需求**：想知道播客哪里可以植入广告

**操作流程**：
1. 粘贴播客URL或文字稿
2. 点击"开始分析"
3. 查看"💰 广告位机会"分类
4. 查看每个机会的预估收益
5. 参考口播文案示例
6. 选择收益最高的位置植入

**预期结果**：
- 识别3-5个广告位机会
- 总预估收益：¥15,000-30,000
- 获得具体的植入文案和建议

### 场景2：风险规避
**用户需求**：担心内容有舆论风险

**操作流程**：
1. 设置较低的风险阈值（如粉丝影响500人）
2. 输入播客内容
3. 重点关注"⚠️ 风险预警"分类
4. 查看风险原因和修改建议
5. 按建议修改敏感内容
6. 重新分析验证

**预期结果**：
- 识别所有潜在风险点
- 获得具体修改建议
- 避免舆论危机和掉粉

### 场景3：内容出圈
**用户需求**：希望内容能够病毒传播

**操作流程**：
1. 输入播客内容
2. 查看"🚀 传播爆点"分类
3. 选择预估影响力最大的爆点
4. 参考推广策略
5. 制作短视频切片
6. 按建议在社交媒体传播

**预期结果**：
- 识别2-4个传播爆点
- 预估影响50,000+粉丝
- 获得具体推广策略

## 五、技术特性

### 1. 双模式输入
- **URL模式**：自动调用网页内容总结API提取文字
- **文字稿模式**：直接使用用户输入，跳过提取步骤

### 2. 结构化输出
- AI返回标准JSON格式
- 前端解析并渲染为卡片
- 支持按类型分组展示

### 3. 本地存储
- 阈值设置保存在localStorage
- 刷新页面后设置保持
- 支持恢复默认值

### 4. 实时反馈
- 流式显示AI分析进度
- Toast提示关键步骤
- 加载状态清晰可见

### 5. 响应式设计
- 桌面端：多列布局，信息密度高
- 移动端：单列布局，触控友好
- 自适应字体和间距

**品牌方**：
- 金钱：20000元
- 粉丝：10000人
- 互动率：10%
- 品牌价值：85分（高度重视品牌形象）

### 2. 分析频率建议

- **录制前**：分析脚本，提前规避风险
- **录制后**：分析成品，优化剪辑和植入
- **发布前**：最终检查，确保万无一失

### 3. 建议优先级

1. **高优先级**：立即处理
   - 高风险预警（潜在损失>10000元或掉粉>5000）
   - 高价值广告位（收益>15000元）
   - 强传播爆点（影响>20000粉丝）

2. **中优先级**：重点关注
   - 中等风险（损失5000-10000元）
   - 中等收益机会（5000-15000元）
   - 内容优化建议

3. **低优先级**：参考即可
   - 低风险提示
   - 小额收益机会
   - 细节优化建议

## 七、常见问题

**Q: AI分析需要多长时间？**
A: 通常30-60秒，取决于播客长度和内容复杂度。

**Q: 预估价值准确吗？**
A: 基于历史数据和行业标准，准确率约70-80%，仅供参考。

**Q: 可以分析多长的播客？**
A: 建议30分钟-2小时，过短内容不足，过长可能超时。

**Q: 支持哪些语言？**
A: 目前仅支持中文播客内容。

**Q: 阈值设置会影响分析结果吗？**
A: 不会。阈值只影响是否弹窗提示，不影响分析内容。

**Q: 可以导出分析报告吗？**
A: 当前版本暂不支持，建议截图保存关键建议。
