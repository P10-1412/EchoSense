# choSense播客分析应用MVP需求文档

## 1. 项目概述

### 1.1 项目名称
choSense 播客分析应用（MVP版本）

### 1.2 核心定位
专注于为商业化与专业化发展的播客创作者提供核心分析功能的简化版应用，以AI驱动为核心，提供高商业价值、高传播潜力节点与高舆论风险点的识别分析，帮助创作者快速获得关键洞察。

### 1.3 解决的核心痛点
- 创作决策难：快速判断内容变现价值和出圈潜力
- 风险规避难：及时发现潜在舆论风险点
- 精力消耗大：简化分析流程，快速获取关键建议

## 2. 核心产品逻辑

### 2.1 核心理念
- 静默分析：系统自动识别关键节点，减少用户操作
- 高价值推送：仅展示前1%的高价值/高风险内容
- 可落地建议：提供具体的操作方案和参考案例

### 2.2 差异化亮点
- 简化输入：支持URL上传和文字稿上传
- 核心模块：商业价值评估、传播价值评估、风险评估
- 快速反馈：快速展示关键分析结果和建议

## 3. 核心功能模块

### 3.1 内容输入模块

#### 3.1.1 输入方式
- URL上传：输入播客RSS链接，系统自动抓取音频转写文本
- 文字稿上传：支持直接上传播客文字内容

#### 3.1.2 处理提示
界面显示：「choSense正在分析您的播客内容，仅展示最有价值的分析结果」

### 3.2 商业价值评估模块

#### 3.2.1 分析目标
识别可商业利用的内容节点，匹配适配的广告形态

#### 3.2.2 Demo呈现

【高商业价值节点】 段落内容：「我最近尝试了一款轻量化的办公软件，不用复杂设置就能实现多设备同步，对于经常出差的创业者来说太友好了」
分析结论：该段内容适合「中插口播广告」「品牌共创测评」两种广告形态
参考案例：类似播客单次合作报价区间为8000-15000元
可操作建议：提供口播广告脚本参考内容

### 3.3 传播价值评估模块

#### 3.3.1 分析目标
识别具备出圈潜力的观点节点，提供传播路径建议

#### 3.3.2 Demo呈现

【高传播潜力节点】 段落内容：「很多人觉得『内卷』是年轻人的专利，但其实中年人的内卷更隐蔽——不是比加班时长，而是比孩子的教育、比父母的养老规划」
分析结论：适合剪辑为45秒传播片段，核心传播路径为「小红书观点共鸣」「知乎话题讨论」「同行播客引用」
可操作建议：提供具体的传播策略建议

### 3.4 舆论风险评估模块

#### 3.4.1 分析目标
预判潜在争议点，提供风险规避建议

#### 3.4.2 Demo呈现

【高风险预警节点】 段落内容：「我觉得所有想创业的人都不用先做商业计划书，浪费时间，直接干就行」
分析结论：该表达易引发「创业认知争议」，可能导致互动率下降
可操作建议：提供缓冲话术参考

### 3.5 分析结果展示模块

#### 3.5.1 展示规则
- 仅显示「历史分位数前1%」的高价值/高风险内容
- 左侧区域展示高商业价值、高传播潜力、高风险预警三类内容
- 右侧区域展示简要的内容结构性总结
- 采用弹窗形式展示关键分析结果和建议

## 4. 设计风格

### 4.1 配色方案
主色调采用深蓝色系（#1E3A8A），辅以科技蓝（#3B82F6）和温和灰（#64748B），体现专业性和科技感

### 4.2 视觉细节
采用现代扁平化设计，圆角半径8px，微妙阴影效果增强层次感，图标风格简约线性

### 4.3 整体布局
左右分栏式布局，左侧为高优先级内容区域，右侧为常规分析结果展示，确保信息层次清晰
