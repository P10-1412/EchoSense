# choSense播客分析应用需求文档

## 1. 应用概述

### 1.1 应用名称
choSense

### 1.2 应用描述
专为寻求商业化与专业化发展的播客创作者打造的AI驱动「创作副驾驶」，秉持「不替代创作、不打断表达」的隐形辅助理念，核心解决创作者在内容是否具备变现价值、是否具有出圈潜力、是否存在舆论与掉粉风险三大关键判断难题。

## 2. 核心功能

### 2.1 内容分析功能
- 播客URL上传：支持用户输入播客网址
- 自动文字提取：从播客中提取文字内容
- AI智能分析：对播客文本与结构进行全面分析

### 2.2 价值判断功能
- 商业化价值评估：识别可商业化的情绪高峰
- 传播价值评估：识别可传播的观点爆点
- 风险评估：检测舆论与掉粉风险

### 2.3 数据驱动决策
- 历史数据对比：结合平台内外历史数据
- 用户反馈分析：纳入用户真实反馈
- 量化价值判断：对内容实际收益水平和风险成本进行数据库驱动的量化评估

### 2.4 智能提示系统
- 静默化运行：仅在关键节点主动提示
- 高价值内容筛选：筛选出显著高于基线的商业或传播价值内容
- 风险预警：提示历史样本中高舆论风险内容
- 相似案例参考：提供相关成功案例参考

### 2.5 结果展示功能
- 弹窗提示：高价值内容通过左侧文本框弹窗显示
- 一般内容展示：一般价值内容在右侧文本框展示
- 轻量体验：其余时间保持静默，保障低侵入的创作体验

## 3. 运行模式

### 3.1 核心理念
- 不替代创作：尊重创作者原创性
- 不打断表达：保持创作流程连贯性
- 隐形辅助：静默化运行，仅在关键节点介入

### 3.2 助力目标
- 更安全地表达：有效规避舆论与掉粉风险
- 更聪明地变现：高效识别商业化机会
- 降低试错成本：量化创作直觉，减少探索成本
- 内容资产化：帮助创作者将内容转化为可盈利资产

## 4. 设计风格

### 4.1 配色方案
主色调采用深蓝色系（#1E3A8A），辅以科技蓝（#3B82F6）和温和灰（#64748B），体现专业性和科技感

### 4.2 视觉细节
采用现代扁平化设计，圆角半径8px，微妙阴影效果增强层次感，图标风格简约线性

### 4.3 整体布局
左右分栏式布局，左侧为高优先级内容区域，右侧为常规分析结果展示，确保信息层次清晰
