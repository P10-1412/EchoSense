# choSense播客分析应用需求文档

## 1. 项目概述

### 1.1 项目名称
choSense 播客分析应用

### 1.2 核心定位
choSense 是专为商业化与专业化发展的播客创作者打造的「静默运行内容价值雷达」，以 AI 驱动为核心，秉持「不替代创作、不打断表达」的隐形辅助理念，精准识别内容中的高商业价值、高传播潜力节点与高舆论风险点，提供全科视角的可落地建议，助力创作者更安全地表达、更聪明地变现、更高效地实现内容资产化。

### 1.3 解决的核心痛点
- 创作决策难：无法精准判断内容是否具备变现价值、出圈潜力
- 风险规避难：难以预判舆论争议、掉粉等潜在风险
- 精力消耗大：创作者需投入大量时间进行内容分析、商业适配，分散创作注意力
- 视角单一化：缺乏法律、商业、心理等多学科专业支撑，易出现认知盲区

### 1.4 项目愿景
成为播客创作者的「决策基础设施」，通过隐形辅助与全科支持，让创作更纯粹、变现更高效、表达更安全，推动播客行业的专业化与商业化升级

## 2. 核心产品逻辑与差异化亮点

### 2.1 核心理念
- 不替代创作：尊重创作者原创主权，不干预内容生成过程
- 不打断表达：静默运行模式，仅在关键节点主动介入，保障创作流程连贯性
- 隐形辅助：以「最小干扰」为原则，节省创作者分析精力，聚焦核心创作
- 全科支持：整合多学科视角，提供专业、可落地的决策参考，填补认知盲区

### 2.2 差异化亮点
- 静默式体验：区别于传统「全量输出报告」的分析工具，仅推送前 1% 高价值信息，极致降低干扰
- 相对价值评估体系：不输出「绝对金额/粉丝量」，而是提供「相对价值+参考案例+可解释路径」，专业且可信
- 全科画像体系：基于法律、心理、商业、健康四大核心学科，建立创作者长期画像，提供个性化、多维度支持
- 可落地导向：所有建议均配套具体操作方案（如口播广告脚本、风险规避话术），而非空泛理论

## 3. 核心功能模块

### 3.1 内容输入与静默分析模块

#### 3.1.1 输入路径（Demo 适配版）
- URL 上传：支持输入播客 RSS 链接，系统自动抓取音频转写文本（Demo 中直接调用预设转写示例，降低技术门槛）
- 文字稿上传：支持上传提前整理的播客文字稿（模拟平台转写结果）
- 回放式分析：除了极高价值内容的弹窗之外，用户点击「回放分析」后，界面以时间轴滚动模拟创作完成后的流程，关键提示逐条「浮现」，而非一次性加载所有结果，还原「静默运行、按需出现」的核心体验

#### 3.1.2 核心话术（强化产品认知）
界面显著位置提示：「choSense 默认静默运行，仅在检测到高价值或高风险节点时出现，不打扰你的创作节奏」

### 3.2 价值评估模块（AI 核心壁垒）

核心原则：不输出绝对价值，仅提供「相对价值+参考案例+可解释路径」，让评估结果具备专业参考意义。只在初筛为中高价值内容（10%的内容或最高的内容）后触发。

#### 3.2.1 商业化价值评估
目标：精准识别「可被商业利用的内容节点」，匹配适配的广告形态与报价参考。

计算路径：
1. 候选段落筛选：通过 AI 识别情绪波动幅度大、观点明确（含「我认为/建议/推荐」等表述）、关联消费/决策/身份相关实体（如产品、服务、生活方式）的段落
2. 商业适配度打分：从「广告自然嵌入可能性」「受众明确度」「观点闭环完整性」三个维度打分，筛选前 1% 高适配段落
3. 参考案例匹配：调用历史数据库，匹配「相似表达结构+相似受众+相似播客体量」的案例，输出适配广告形态与报价区间

Demo 呈现示例：

【高商业价值节点】 段落内容：「我最近尝试了一款轻量化的办公软件，不用复杂设置就能实现多设备同步，对于经常出差的创业者来说太友好了」
分析结论：该段内容在历史样本中，最常适配「中插口播广告」「品牌共创测评」两种形态
参考案例：粉丝量 5-10 万的职场类播客，同类内容单次合作报价区间为 8000-15000 元
可操作建议：口播广告脚本参考——「经常出差的朋友应该都懂多设备同步的痛点，我最近在用的XX办公软件就解决了这个问题，不用复杂设置，打开就能用，现在新用户注册还能领 30 天会员，感兴趣的可以去评论区看链接」

#### 3.2.2 传播价值评估
目标：识别「具备出圈潜力的观点节点」，提供传播路径建议。

核心指标：观点反直觉程度、与主流叙事的冲突性、语言可切片性（是否适合 30-60 秒截取传播）。

Demo 呈现示例：

【高传播潜力节点】 段落内容：「很多人觉得『内卷』是年轻人的专利，但其实中年人的内卷更隐蔽——不是比加班时长，而是比孩子的教育、比父母的养老规划」
分析结论：该段内容结构适合剪辑为 45 秒传播片段，历史样本中同类内容的核心传播路径为「小红书观点共鸣」「知乎话题讨论」「同行播客引用」
可操作建议：1. 截取该段落作为短视频素材，搭配文字「中年人的隐蔽内卷，你中招了吗？」发布至社媒；2. 围绕「中年内卷的隐蔽形式」展开专题讨论，吸引垂直受众互动。

#### 3.2.3 舆论与掉粉风险评估
目标：预判潜在争议点，提供风险规避建议与历史案例参考。

核心指标：观点极端度、事实不确定性、群体标签触及度（是否涉及敏感群体）、历史争议话题相似度。

Demo 呈现示例：

【高风险预警节点】 段落内容：「我觉得所有想创业的人都不用先做商业计划书，浪费时间，直接干就行」
分析结论：该表达在历史样本中，最易引发「创业认知争议」，导致评论区极化，相似播客曾出现 3-8% 的互动率下降
可操作建议：补充缓冲话术——「当然这只是我的个人经验，适合小成本试错的轻资产创业，如果你是重资产项目，还是建议做好商业计划书和风险评估，根据自己的情况调整」

### 3.3 全科画像与多视角建议模块

聚焦「法律、心理、商业、健康、传播」五大高价值学科，建立创作者长期画像，记录核心信息并提供个性化建议，形成可追溯的学科记录文档。同时，基于用户的播客主题，可以灵活添加1~2个主题学科画像（但要有上限不能无限扩张）。

#### 3.3.1 五大核心学科画像定义

- 法律画像：记录播主在多轮播客中出现的潜在违法表达、事实性风险、责任归属模糊表述（如绝对化广告宣传、不实信息传播、侵权风险等）
- 心理画像：记录播主反复出现的信念模式、情绪叙事特点、潜在认知偏差（如极端完美主义、过度焦虑、小众观念等）
- 商业/会计画像：记录播主在创业、投资、职场分享中出现的认知误区、低级错误（如现金流概念混淆、投资风险忽视、商业模式漏洞等）
- 健康/医学画像：记录播主描述的身体状态、生活习惯，提示潜在健康风险（不做疾病诊断，仅提示「需专业检查的信号」）
- 传播学画像：用户的账号风格、受众特征，用于辅助价值评估。不同账号发表相同观点的价值是不同的。必须在价值评估中考虑到这一点

#### 3.3.2 学科记录文档格式

【法律画像记录】 日期：2025-03-18 播主：XXX 主题：创业避坑指南
核心记录：1. 提及「这款产品绝对能帮你月入过万」，存在绝对化广告宣传风险；2. 引用某企业案例时未注明来源，存在侵权风险
建议：1. 将绝对化表述修改为「部分用户使用后实现月入过万，具体因人而异」；2. 补充案例来源「该案例来自 XX 企业公开报道」

【心理画像记录】 日期：2025-03-18 播主：XXX 主题：职场压力应对
核心记录：反复强调「如果做不到完美，这件事就没必要做」，呈现明显完美主义倾向，多次因「担心不完美」延迟项目推进
建议：尝试「完成比完美更重要」的行动原则，针对重要任务设定「80 分标准」，先完成再优化，减少拖延内耗

### 3.4 智能提示与结果展示模块

#### 3.4.1 提示规则
仅触发「历史分位数前 1%」的高价值/高风险内容：将当前内容指标纳入历史数据库，通过相对排名确定是否触发提示，确保推送内容的稀缺性与价值感。

#### 3.4.2 界面布局（低干扰设计）
- 左侧（高优先级区域）：仅展示高商业价值、高传播潜力、高风险预警三类内容，以弹窗形式浮现，点击可查看详细分析与建议
- 右侧（常规区域）：展示内容结构性总结、全科画像历史记录，默认折叠，用户可按需展开查看
- 整体体验：无提示时界面简洁，仅保留核心操作按钮，保障「低侵入」创作体验

## 4. 设计风格

### 4.1 配色方案
主色调采用深蓝色系（#1E3A8A），辅以科技蓝（#3B82F6）和温和灰（#64748B），体现专业性和科技感

### 4.2 视觉细节
采用现代扁平化设计，圆角半径8px，微妙阴影效果增强层次感，图标风格简约线性

### 4.3 整体布局
左右分栏式布局，左侧为高优先级内容区域，右侧为常规分析结果展示，确保信息层次清晰
